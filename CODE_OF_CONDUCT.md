# Code of Conduct

## NVO987 Research Infrastructure

The NVO987 project is an independent, decentralized research infrastructure 
focused on identity systems, knowledge representation, and open web technologies.

This repository is part of that infrastructure and follows a set of principles 
to ensure a constructive, respectful, and transparent environment.

---

## 1. Purpose

The goal of this project is to develop and maintain an open, interoperable, 
and machine-readable research ecosystem based on decentralized identity 
and semantic web standards.

All contributions, discussions, and interactions should support this objective.

---

## 2. Expected Behavior

Participants are expected to:

- Communicate clearly and respectfully
- Focus on technical, research, or conceptual contributions
- Provide constructive feedback
- Respect the decentralized and open nature of the project
- Maintain a professional tone in all interactions

---

## 3. Unacceptable Behavior

The following behaviors are not tolerated:

- Harassment, discrimination, or offensive language
- Personal attacks or hostility
- Disruptive or irrelevant contributions
- Misrepresentation of identity or intent
- Attempts to compromise the integrity of the infrastructure

---

## 4. Research Integrity

This project emphasizes:

- Transparency of methods and data
- Verifiability of identity and sources
- Traceability of contributions
- Respect for intellectual work and attribution

Contributors should ensure that all submitted content aligns with these principles.

---

## 5. Decentralized Identity Context

This repository is connected to the decentralized identifier:

did:web:identity.nvo987.us

All infrastructure components ultimately reference this identity root. 
Contributions should respect the integrity and consistency of this system.

---

## 6. Scope

This Code of Conduct applies to:

- Source code
- Documentation
- Issues and discussions
- Pull requests
- Any related communication channels

---

## 7. Enforcement

Project maintainers reserve the right to:

- Remove or edit content that violates these guidelines
- Reject contributions that do not align with the project goals
- Restrict or block participation in case of repeated violations
