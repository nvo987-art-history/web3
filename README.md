# NVO987 Web3 Node

This repository contains the web3 layer of the NVO987 decentralized research infrastructure.

It is part of a distributed system connecting identity, knowledge representation, and machine-readable research environments through open web standards.

---

## 🌐 Identity Root

The entire infrastructure is anchored in the decentralized identifier:

**did:web:identity.nvo987.us**

This DID serves as the cryptographic and semantic root of identity across all nodes.

---

---

## 🧠 Purpose of this Repository

This repository hosts the **web3-facing layer** of the infrastructure.

It is intended to:

- provide a decentralized web entry point  
- integrate with distributed storage systems (IPFS, ENS, etc.)  
- expose machine-readable identity and infrastructure references  
- support future interoperability with web3 environments  

---

## ⚙️ Technical Scope

The system is based on open standards, including:

- Decentralized Identifiers (DID)  
- JSON-LD  
- RDF / Linked Data  
- FOAF  
- RSS / Atom  

These technologies enable interoperability between human-readable and machine-readable research systems.
